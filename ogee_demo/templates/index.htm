{% blocktrans %} Ogee Arches is a complete package for the Arches platform that implements the Linked.Art data model with pre-built vocabulary and full documentation. {% endblocktrans %}

{% blocktrans %} Created by Getty Digital with Takin.solutions, Ogee provides the complete set of Linked.Art models configured and ready to use — taking the difficulty out of starting a Linked.Art project on Arches. {% endblocktrans %}

{% trans "02 — Foundation" %}

{% trans "Standardized Data Models" %}

{% trans "Fully implemented Linked.Art semantic data models" %}

{% blocktrans %} The Linked.Art community creates standardized semantic data models using the CIDOC CRM standard — enabling accurate recording of cultural heritage objects and their relationships, integrated across institutions. {% endblocktrans %}

{% trans "Ogee strictly follows the Linked.Art modeling patterns (current as of December 2023) and implements all eleven projected data models. Users start making Linked.Art-compatible data from day one, with the freedom to extend or limit models to their use case." %}

{% trans "03 — Reference Data" %}

{% trans "Defined Vocabulary" %}

{% trans "Pre-organized reference data to get you started" %}

{% trans "Rich semantic models need rich vocabulary — and building correct vocabulary structures from scratch is one of the biggest delays in implementation." %}

{% blocktrans %} Ogee ships with a ready-made vocabulary in the RDM, primed with sample terms from the Getty AAT for all standardized concept types. You get a clear guide to the intention of each semantic field, with good default terms you can extend or prune. {% endblocktrans %}

{% trans "04 — Knowledge" %}

{% trans "Full Semantic Documentation" %}

{% trans "Modular documentation that explains your system semantics" %}

{% blocktrans %} Ogee is documented using Pletka, which creates modular documentation of every model and field — a shared reference for modellers, developers, and end users. {% endblocktrans %}

{% trans "The documentation is open: adopt it, extend it, modify it using the semantic reference data modelling method, and share your Arches implementation with others." %}

{% trans "05 — Philosophy" %}

{% trans "Why Ogee?" %}

{% trans "Patterns that show the unity across diversity" %}

{% trans "The ogee is a design pattern that appears across cultures and centuries — in Islamic, Judaic, Christian, European, and contemporary art. Characterized by its double-S curve, it tessellates beautifully, linking across potentially infinite space." %}

{% trans "It captures the spirit of Linked.Art: common patterns adopted across institutions and disciplines, enabling infinite variation while serving as a framework for integrating knowledge into a wider fabric." %}

{% trans "Ogee Arches is a starting point, not an end point — a path into the semantic world that we hope will open up a new mosaic of information between its adopters." %}
